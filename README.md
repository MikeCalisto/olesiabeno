# olesiabeno

Landing page for the online course "Розбір гардеробу" by Олеся Beno.

## Структура

- `index.html` — головний лендінг
- `privacy.html` — політика конфіденційності
- `offer.html` — публічна оферта
- `photo1.jpg … photo6.jpg` — фото образів клієнток
- `author.jpg` — фото автора
- `before.png` / `after.png` — шафа до/після курсу

Статичний HTML, без build-кроку. Деплой — Vercel.
